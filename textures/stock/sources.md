Stock Texture File Sources
-

* Most stock textures are from KittopiaTech Dumps: https://github.com/Kopernicus/kittopia-dumps/tree/master
* Jool color map is from the Kerbal Maps project: https://forum.kerbalspaceprogram.com/topic/23154-022-kerbal-maps-web-maps-of-all-bodies-biomes-and-text-info-v011-oct-8/
* Jool height map is assumed to be pure black
* Normal maps have been calculated from the height maps using a custom script. Beware of artefacts.
